# E-Commerce Website

A modern and responsive e-commerce website that allows users to browse products, add items to a shopping cart, and complete purchases online. Built to showcase frontend and full-stack development skills.

## Features
- Browse and search products
- Product categories and filtering
- Add products to shopping cart
- View cart and checkout simulation
- Responsive design for mobile and desktop
- Contact form for customer inquiries

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project demonstrates the ability to build a functional online store with interactive features, clean UI, and responsive design. It showcases skills in frontend development and, optionally, backend integration.

## Future Improvements
- Integrate real payment system
- Add user authentication and profiles
- Implement backend for product database management
- Improve product filtering and search
